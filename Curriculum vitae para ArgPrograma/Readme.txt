La funcionalidad de este proyecto fue crear un CV que aparte de diseñarlo con bootstrap 
y mantenerlo responsive, almacene un número en el local storage del navegador que incluso
pueda copiarse mediante un botón. Ademas tiene un pequeño formulario para simular la
recepción de datos que los campos sean obligatorios y uso de caracteres limitados. Finalmente
incluí algunas alertas utilizando funciones en javascript.
